Esta es una landing page de alto rendimiento diseñada para el **Ing. Onel Martinez**, profesional de la **Universidad Tecnológica de Panamá (UTP)**. El objetivo principal es la captación de estudiantes de nivel secundaria y universitario que buscan excelencia en Matemáticas, Física y Química.

## 🚀 Características Principales

* **Diseño Premium & Mobile First:** Optimizado para que las "cards" y secciones respiren en dispositivos móviles, evitando saturación visual.
* **SEO Avanzado:** Implementación de metadatos Open Graph, JSON-LD (Schema.org) para resultados enriquecidos y jerarquía de encabezados (H1-H3).
* **UX Orientada a Conversión:** Call to Action (CTA) estratégico en color `emerald-500` para maximizar los clics hacia WhatsApp.
* **Componentes Modernos:** * Hero Slider con transiciones suaves.
    * Sistema de iconos dinámicos con **Lucide Icons**.
    * Efectos visuales de desenfoque (backdrop-blur) y gradientes de cristal.

## 🛠️ Tecnologías Utilizadas

* **HTML5 / CSS3** (Semántico para SEO).
* **Tailwind CSS** (Framework de utilidades para un diseño ultra-rápido).
* **Lucide Icons** (Iconografía vectorial minimalista).
* **Google Fonts** (Tipografías *Poppins* e *Inter*).

## 📁 Estructura del Código

- `index.html`: Archivo principal con toda la estructura semántica.
- `SEO Blocks`: Metaetiquetas y datos estructurados para Google.
- `Scripts`: Lógica para el slider del Hero y la hidratación de iconos.

## 📈 Optimización Realizada

1.  **Contraste Visual:** Ajuste de secciones para evitar colisiones de color entre el cuerpo y el footer.
2.  **Accesibilidad:** Uso de etiquetas `alt` descriptivas y roles de navegación.
3.  **Responsive Design:** Sistema de márgenes y paddings dinámicos para que el contenido no toque los bordes físicos de la pantalla en móviles.
